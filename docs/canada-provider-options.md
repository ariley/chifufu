# Canada Provider Options

Canada should be treated as a later provider rollout after the US provider layer is stable.

## Best Leads

- Instacart Developer Platform: likely the best US/Canada bridge if access includes retailer catalog, availability, images, and store context.
- Savvi Prices: Canada-focused grocery data API that advertises store coverage, sale flags, availability, and matching confidence.
- Direct retailer feeds: Loblaw, Sobeys, Metro, Save-On-Foods, Walmart Canada, Costco Canada, and T&T would require partnership access or a data vendor.

## Less Useful As Primary Sources

- Walmart Canada Marketplace APIs are seller/partner APIs, not a broad consumer price-comparison API.
- PC Optimum is loyalty/offers oriented, not an open inventory and price API.
- Statistics Canada food price data is useful for analytics, not live store-level product search.

## Product Rule

Canada should follow the same product rule as the US: Google Places or maps data can discover stores and reviews, but must not be used to invent product availability or prices.
